## 1ï¸âƒ£ Assembler (`src/Assembler.py`)

A Python-based assembler that converts **RISC-V RV32I assembly** into
**32-bit machine code**.

### âœ” Features

-   Supports all major RV32I formats: **R, I, S, B, J**
-   Handles:
    -   Register ABI encoding\
    -   Two's complement immediates\
    -   Label resolution\
    -   PC-relative addressing
-   Outputs machine code ready for simulation

### âœ” Usage

``` bash

```

## 2ï¸âƒ£ Simulator (`src/Simulator.py`)

### âœ” Features

-   Logs execution state each cycle
