#  RISC-V Assembler & Simulator (Python)

This repository contains a complete **RV32I assembler and simulator**,
implemented from scratch in Python.\
It also includes an **automated testing framework** for validating
correctness of both the assembler and the simulator.

The project is designed for educational use --- particularly for courses
in **Computer Architecture**, **Instruction Set Design**, and
**Machine-Level Execution**.

#  Project Components

##  Assembler (`src/Assembler.py`)

A Python-based assembler that converts **RISC-V RV32I assembly** into
**32-bit machine code**.

###  Features

-   Supports all major RV32I formats: **R, I, S, B, J**
-   Handles:
    -   Register ABI encoding\
    -   Two's complement immediates\
    -   Label resolution\
    -   PC-relative addressing
-   Outputs machine code ready for simulation

###  Usage

``` bash
python src/Assembler.py
```

------------------------------------------------------------------------

##  Simulator (`src/Simulator.py`)

Executes machine-code output generated by the assembler.

###  Features

-   Decodes & executes RV32I instructions
-   Maintains:
    -   Register file
    -   Program Counter
    -   Memory map
-   Logs execution state each cycle

###  Usage

``` bash
python src/Simulator.py input_binary.txt output_log.txt
```

#  Automated Testing Framework

Located in:

    Automated Testing/automatedTesting/

### Includes:

-   **AsmGrader.py** --- assembler correctness checker\
-   **SimGrader.py** --- simulator correctness checker\
-   **tests/** --- reference binaries, error tests, and expected outputs

Run all tests:

``` bash
python Automated Testing/automatedTesting/src/main.py
```

#  Supported RV32I Instructions

### **R-Type**

`add`, `sub`, `and`, `or`, `xor`, `slt`, `srl`

### **I-Type**

`addi`, `lw`, `jalr`

### **S-Type**

`sw`

### **B-Type**

`beq`, `bne`, `blt`

### **J-Type**

`jal`
